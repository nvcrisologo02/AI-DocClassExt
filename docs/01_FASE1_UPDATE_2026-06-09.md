### AB#99372 — Audit Trail Tipologías (Done ✅)

**Ubicación:** `src/backend/DocumentIA.Data/Repositories/TipologiaRepository.cs`

**Implementación completada:**

```csharp
// Métodos con snapshots before/after
public async Task PublicarAsync(int id, string publicadaPor)
{
    var tipologia = await _context.Tipologias.FindAsync(id);
    var before = CloneForAudit(tipologia);  // Snapshot
    tipologia.Estado = EstadoTipologia.Published;
    tipologia.PublicadaEn = DateTime.UtcNow;
    tipologia.PublicadaPor = publicadaPor;
    tipologia.VersionPublicada = tipologia.Version;
    tipologia.FechaActualizacion = DateTime.UtcNow;
    await _context.SaveChangesAsync();
    await AddAuditAsync(tipologia.Id, "Published", publicadaPor, before, tipologia);
}
```

**Base de Datos:**
- Tabla: `TipologiaConfigAudit` (Id, TipologiaId, Accion, Usuario, FechaHora, DetallesJson)
- Migration: `20260424145132_AddTipologiaConfigAudit.cs` (deployada)
- Índices: TipologiaId + FechaHora (ordenado desc)

**Validación:**
- ✅ Tests passing (audit trail functionality)
- ✅ JSON snapshots capturando estado before/after
- ✅ Recuperable vía `TipologiaConfigAuditRepository.GetByTipologiaIdAsync()`

**API Endpoint:**
